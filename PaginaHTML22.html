Hyper Text Markup Lenguage







HTML significa "Hypertext Markup Language" (Lenguaje de Marcado de Hipertexto) y es el lenguaje de marcado estándar utilizado para crear páginas web y aplicaciones web.

HTML (Hypertext Markup Language) fue creado en 1989 por Tim Berners-Lee mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear) en Suiza.

HTML ha recorrido un largo camino desde su creación en 1989. En sus primeras versiones, HTML se centraba en proporcionar una manera básica de estructurar y dar formato al contenido en línea. A medida que se desarrollaba la web, se introdujeron nuevas versiones de HTML con características y mejoras adicionales.






Las etiquetas HTML son elementos que se utilizan para marcar o definir contenido en una página web. Estas etiquetas están compuestas por una etiqueta de apertura y una de cierre, que encierran el contenido que se quiere definir.

Un documento HTML se estructura utilizando las etiquetas HTML para delimitar y organizar el contenido.

HTML y CSS son dos lenguajes diferentes, pero trabajan juntos para crear páginas web y aplicaciones web visualmente atractivas y bien estructuradas.

¿QUE ES HTML?
HTML son las siglas en inglés para Hypertext Markup Language, que se traduce como lenguaje de marcación de hipertexto.

A pesar de tener un nombre complicado, el HTML no es más que un lenguaje usado para crear páginas webs por medio de marcadores (tags) y atributos, que definen cómo el contenido va a ser presentado en un navegador web.

La estructura de HTML es bastante sencilla: las páginas web se componen de elementos HTML, que se definen mediante etiquetas. Las etiquetas se utilizan para definir la estructura de la página, así como para especificar el contenido de la misma.
Historia de HTML
HTML fue creado por primera vez por Tim Berners-Lee, Robert Cailliau y otros a partir de 1989. Son las siglas de Hyper Text Markup Language.
Hipertexto significa que el documento contiene enlaces que permiten al lector saltar a otros lugares del documento oa otro documento por completo. La última versión se conoce como HTML5.
Un lenguaje de marcado es una forma en que las computadoras se comunican entre sí para controlar cómo se procesa y presenta el texto. Para hacer esto, HTML usa dos cosas: etiquetas y atributos.
Versiones de HTML
HTML 1.0: Fue la primera versión de HTML lanzada en 1993, que incluía un conjunto básico de etiquetas para estructurar y dar formato a las páginas web.

HTML 2.0: Fue lanzada en 1995, y agregó nuevas características como formularios y tablas.

HTML 3.2: Fue lanzada en 1997, e incluyó más características y mejoras de estilo.

HTML 4.01: Fue lanzada en 1999, y proporcionó aún más características y mejoras, incluyendo la capacidad de crear hojas de estilo en cascada (CSS).

XHTML: Fue introducido en 2000, y es una versión más estricta y basada en XML de HTML. XHTML fue diseñado para ser más fácil de analizar y escribir para las computadoras, lo que lo hizo más compatible con los estándares web.

HTML5: Es la versión más reciente de HTML, lanzada en 2014, y proporciona muchas nuevas características, como elementos multimedia, formularios avanzados, compatibilidad móvil y semántica mejorada. HTML5 es la versión de HTML más utilizada en la actualidad.
Principales etiquetas de HTML
Hay muchas etiquetas HTML que se pueden utilizar para estructurar y dar formato al contenido de una página web, pero aquí hay una lista de algunas de las etiquetas más comunes y útiles: html>: Define el comienzo y el final del documento HTML.

head>: Define el encabezado de la página web y contiene información meta y otros elementos.

title>: Define el título de la página web, que se muestra en la pestaña del navegador.

body>: Define el cuerpo principal de la página web.

h1> a h6>: Define los encabezados de la página web, donde h1> es el encabezado más grande y h6> es el encabezado más pequeño.

p>: Define un párrafo de texto.

a>: Define un enlace a otra página web o a un elemento dentro de la misma página web.

img>: Define una imagen que se muestra en la página web.

ul> y li>: Define una lista sin orden o una lista con viñetas.

Estas son solo algunas de las etiquetas HTML más comunes, y hay muchas otras que se pueden utilizar para crear páginas web ricas en contenido y bien estructuradas.
Estructura Documento HTM
Hemos visto muchos elementos, pero ahora vamos a saber cómo ponerlos dentro de un código HTML. Al igual que cualquier estándar web, debe seguir algunas reglas para crear estos códigos.


Comenzamos con la declaración ! DOCTYPE html para informar al navegador web que el documento en cuestión es un HTML.


Luego abrimos un tag html, que tendremos que cerrar al final. En su interior agregamos head, que debe contener los metadatos de la página (datos sobre datos), como el título que aparecerá en la pestaña del navegador (title), llamadas a archivos externos y otras que son cruciales para un trabajo de SEO, como las meta tags.


Es importante incluir la tag meta charset = ”utf-8 ″ para que el navegador muestre correctamente ciertos caracteres que usamos en español, como los acentos o la virgulilla.


Finalmente, abrimos el tag body, donde agregamos la parte del documento HTML que contiene los elementos visibles para el usuario.
Cual es la relacion entre HTML y CSS
Ya entendemos que HTML es la estructura de una página web, pero, ¿cómo formatear el estilo de sus elementos?

Para configurar fuentes, colores, alineación, entre otros, la forma como se presentarán los componentes HTML, usamos el CSS, que significa Cascading Style Sheets u hojas de estilo en cascada.

Aunque es posible describir los estilos dentro del propio código HTML, se recomienda que el HTML realice una llamada a uno o más archivos CSS individuales. Esto hace que el código sea más organizado ya que aísla la estructura del estilo.

El CSS se basa en selectores y declaraciones con propiedades y valores. Es decir, elegimos los elementos HTML a los que queremos aplicar los estilos y declaramos cuáles serán las propiedades de estos componentes.
